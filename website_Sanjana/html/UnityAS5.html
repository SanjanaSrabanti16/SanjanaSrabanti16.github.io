Computer Architecture and Cats - Multiplayer Networked Unity Game

Game Design and Development Assignment, February 2021

Description

Brief game idea: Cats trying to break out their friend. They are collecting balls while doing it. It turns into a friendly competition, the cat with the highest score wins.
Player Interaction Pattern: cooperative play, unilateral competition
Exec Summary: the game is based around a age range of young children to teens. This Game has themes sorrounding the move UP, kid shows such as Teen Titan. The themes surround an urban area.
Objective: Get the highScore, Balls are scattered around the environment. Highest score wins!
